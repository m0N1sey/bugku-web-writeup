# BugKu Web3 - alert

## 题目信息
- **平台**: [BugKu CTF](https://ctf.bugku.com/)
- **类型**: Web
- **难度**: ⭐
- **金币**: 1
- **分数**: 10

## 题目描述
打开题目后，页面不断弹出 alert 对话框，无法查看内容。

## 解题过程

### 1. 打开题目
页面不断弹出 alert，点击确定后继续弹。

### 2. 阻止弹窗
勾选浏览器弹窗中的"阻止此页面显示更多对话框"，然后点击确定。

### 3. 查看源码
按 Ctrl + U 查看页面源代码，在源码最后发现 HTML 实体编码：

```html
&#102;&#108;&#97;&#103;&#123;&#49;&#52;&#99;&#102;&#97;&#57;&#102;&#98;&#99;&#54;&#98;&#99;&#50;&#99;&#97;&#49;&#100;&#98;&#101;&#50;&#48;&#56;&#99;&#99;&#102;&#53;&#98;&#54;&#52;&#52;&#48;&#51;&#125;
```
### 4.解码
这是 ASCII 码的 HTML 实体编码，使用在线工具进行 HTML 解码获取flag
