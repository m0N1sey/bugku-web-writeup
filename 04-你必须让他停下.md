# BugKu Web4 - 你必须让他停下

## 题目信息
- **平台**: [BugKu CTF](https://ctf.bugku.com/)
- **类型**: Web
- **难度**: ⭐⭐
- **金币**: 1
- **分数**: 10

## 题目描述
打开题目后页面不断自动刷新，flag 在某一帧的源码中闪现。

## 解题过程

### 1. 打开题目
页面不断刷新，内容变化很快。

### 2. 禁用 JavaScript
按 F12 打开开发者工具，按 F1 进入设置，勾选"Disable JavaScript"。

### 3. 查看源码
刷新页面后不再自动刷新，按 Ctrl + U 查看源码。

### 4. 搜索 flag
在源码中搜索 `flag`，找到隐藏的 flag。
