# BugKu Web5 - GET

## 题目信息
- **平台**: [BugKu CTF](https://ctf.bugku.com/)
- **类型**: Web
- **难度**: ⭐
- **金币**: 1
- **分数**: 10
- **题目链接**: http://123.206.87.240:8002/get/

## 题目描述
页面提示需要用 GET 方式提交参数。

## 解题过程

### 1. 打开题目
页面显示 PHP 代码，提示需要传入 `what` 参数。

### 2. 构造 GET 请求
在 URL 后面添加查询参数：?what=flag

### 3. 提交访问
页面返回 flag
