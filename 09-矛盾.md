# BugKu Web9 - 矛盾

## 题目信息
- **平台**: [BugKu CTF](https://ctf.bugku.com/)
- **类型**: Web
- **难度**: ⭐⭐
- **金币**: 1
- **分数**: 10
- **题目链接**: http://123.206.87.240:8002/get/index1.php

## 题目描述
题目要求传入一个参数 `num`，要求既**不是数字**又**等于 1**，看似矛盾。

## 解题过程

### 1. 分析源码逻辑

题目源码大致如下：

```php
$num=$_GET['num'];
if(!is_numeric($num)) {      // 要求 num 不是数字
    echo $num;
    if($num==1) {            // 但又要求 num 等于 1
        echo 'flag{**********}';
    }
}
